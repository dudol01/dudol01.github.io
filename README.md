## 1. 학급 경영 지원 도구
담임 교사의 반복적인 학급 운영 업무를 자동화하여 효율성을 높이는 프로그램 모음.

### ✅ [자리 바꾸기 (Seating Arrangement)](https://dudol01.github.io/seatingarrangement.html)
* **목적:** 학급 내 학생 좌석 배치를 공정하고 신속하게 수행
* **주요 기능:**
    * 전체 학생 대상 무작위(랜덤) 좌석 배정
    * 성별 고려 또는 특정 조건 반영 가능(설정에 따름)
    * 배정 결과 즉시 화면 출력 및 확인
* **[실행하기]** (https://dudol01.github.io/seatingarrangement.html)

### ✅ [모둠 편성기 (Smart Grouper)](https://dudol01.github.io/smartgrouper)
* **목적:** 협동 학습 및 프로젝트 수업을 위한 조(Group) 편성
* **주요 기능:**
    * 전체 인원을 원하는 모둠 수로 자동 분할
    * 모둠별 인원 균등 배분 알고리즘 적용
    * 빠른 재편성 기능을 통해 수업 준비 시간 단축
* **[실행하기]** (https://dudol01.github.io/smartgrouper)

### ✅ [1인 1역 로테이터 (Role Rotater)](https://dudol01.github.io/rolerotater.html)
* **목적:** 학급 청소 구역 및 1인 1역 역할 순환 관리
* **주요 기능:**
    * 주/월 단위 역할 변경 시 자동 순환 배정
    * 이전 역할과 겹치지 않도록 다음 순번 자동 지정
    * 학급 구성원 전체가 다양한 역할을 경험하도록 유도
* **[실행하기]** (https://dudol01.github.io/rolerotater.html)
